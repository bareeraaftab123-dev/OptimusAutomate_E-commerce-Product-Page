## 📌 Project Overview

**E-Commerce Product Page** is a fully responsive product display page designed for an online store. The page showcases a single product with detailed information, pricing, customer reviews, and an interactive shopping experience.

This project was developed as part of my Web Development Internship at **Optimus Automation**, demonstrating proficiency in frontend development, responsive design, and interactive user interfaces.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Product Display** | High-quality product image gallery with thumbnails |
| **Product Details** | Name, price, description, ratings, and availability |
| **Size/Color Selection** | Interactive options for product variants |
| **Quantity Selector** | Increase/decrease quantity with cart functionality |
| **Add to Cart** | Interactive button with visual feedback |
| **Customer Reviews** | Display of user reviews and ratings |
| **Related Products** | Suggested products section |
| **Responsive Design** | Optimized for desktop, tablet, and mobile devices |
